# The Trivial Pursuit of a Startup Idea

A 90-minute, single-screen facilitation game for **3 co-founders** (CEO · CTO · CCO)
hunting for a fundable **AI-first** startup idea.

Win by completing the pie: a fundable idea needs all six wedges —
🔵 Market · 🩷 Pain · 🟢 AI capability · 🟡 Why now · 🟤 Unfair insight · 🟠 The win.

- **Lap 1 — Diverge:** scatter problems across all six categories (die roller, prompt deck, sticky wall, trivia jolt).
- **Pick contenders:** dot-vote, choose 2–3 candidate ideas.
- **Lap 2 — Play for the pie:** take ONE idea, earn all six wedges (QM gating, gap-recording, live filling pie).
- **Synthesis:** auto-filled winner statement, print/PDF.

Pure static site — just `index.html`. No backend, state saved to `localStorage`.

## Run locally
Open `index.html` in any browser. That's it.

## Deploy
Static site, zero config. On Vercel it serves `index.html` automatically.
